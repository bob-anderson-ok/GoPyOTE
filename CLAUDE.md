## Project Overview

GoPyOTE is a Go desktop application for astronomical occultation timing and diffraction analysis. It provides interactive light curve visualization, diffraction light curve fitting with NCC alignment, Monte Carlo uncertainty estimation, and VizieR export. It integrates with an external diffraction calculation utility (IOTAdiffraction.exe).

## Build and Run Commands

```bash
go build                 # Build the binary (GoPyOTE.exe on Windows)
go mod download          # Download dependencies
./GoPyOTE.exe            # Run the application
```

Requires Go 1.24+.

## Architecture

The application is a multi-file structure with these key components:

### Source Files
- **main.go**: Primary application file — UI setup, CSV loading, startup overlay, IOTAdiffraction integration, parameter dialogs, plot interactions
- **fit.go**: Fit/Monte Carlo logic — NCC sliding fit, path offset search, plot generation (overlay, NCC, path offset, histograms), result display
- **plot_widget.go**: Custom `LightCurvePlot` Fyne widget for interactive plot rendering with gonum/plot
- **logging.go**: Action log file creation and timestamped logging
- **vizier.go**: VizieR .dat file export, zip packaging, NA spreadsheet import
- **types.go**: Shared data types (`OccultationParameters`, `EllipseParams`, etc.)
- **lightcurve/**: Package for light curve extraction, path computation, edge detection, image loading

### Key Global Variables
- `appDir`: Directory containing the executable — used to resolve all relative file paths (diffraction images, IOTAdiffraction.exe) regardless of OS working directory
- `resultsFolder`: Path to the `-RESULTS` folder created in `appDir` when a CSV is opened — all output files (plots, logs, VizieR copies) are written here
- `lastDiffractionParamsPath`: Parameters file used for the last IOTAdiffraction run (persisted in preferences)
- `lastDiffractionTitle`: Title from that parameters file (used in plot titles and window title bars)
- `lastLoadedParamsPath`: Parameters file from the parameter entry dialog

### Custom Light Curve Widget
- `LightCurvePlot`: Custom Fyne widget for interactive plot rendering
- `PlotPoint`/`PlotSeries`: Data structures for plot points and series
- `lightCurvePlotRenderer`: Renders plots using gonum/plot, converts to PNG, displays in Fyne canvas
- Supports interactive point selection with coordinate conversion between screen and data space

### Windows API Integration
- Direct syscall to user32.dll for window positioning
- Persists window geometry across sessions via Fyne preferences

### External Process Integration
- Spawns IOTAdiffraction.exe (resolved via `appDir`) and streams its output
- Uses goroutines for stdout/stderr reading with mutex-protected shared state
- Thread-safe UI updates via `fyne.Do()`

### UI Structure
- Fyne v2 application with tabbed interface
- HSplit layout: tabs on left, interactive plot on right
- Menu bar with File, Edit, Help menus
- Startup overlay shows diffraction image and parameters info (hidden when data loads)
- Parameter entry dialog with dirty-check confirmation on Cancel

### Results Folder Pattern
When a CSV file is opened, a `-RESULTS` folder is created in `appDir` (e.g., opening `mydata.csv` creates `mydata-RESULTS/`). Output files written there include:
- `*.log` — session action log
- `fitPlot.png` — fit result overlay plot
- `observationPath.png` — best-fit observation path on diffraction image
- `searchRange.png` — observation path search range preview
- `peakNCCvsPathOffset.png` — peak NCC vs path offset plot
- `baselineNoiseHistogram.png` — baseline noise histogram
- VizieR `.dat` file copies

## Key Dependencies

- **Fyne v2**: Cross-platform GUI framework
- **gonum/plot**: Scientific plotting library for light curves
- **go-json5**: JSON5 parsing for parameters files
- **excelize**: Excel spreadsheet reading for NA report form import

## Generated Files

- `targetImage16bit.png`, `geometricShadow.png`, `diffractionImage8bit.png`: Generated by IOTAdiffraction.exe in `appDir`
- Parameters files: JSON5 configuration with asteroid/occultation geometry
