## Project Overview

GoPyOTE is a Go desktop application for astronomical occultation timing and diffraction analysis. It provides interactive light curve visualization and integrates with an external diffraction calculation utility (IOTAdiffraction.exe).

## Build and Run Commands

```bash
go build                 # Build the binary (GoPyOTE.exe on Windows)
go mod download          # Download dependencies
./GoPyOTE.exe            # Run the application
```

Requires Go 1.24+.

## Architecture

The application is a single-file structure (`main.go`) with these key components:

### Custom Light Curve Widget (lines 68-358)
- `LightCurvePlot`: Custom Fyne widget for interactive plot rendering
- `PlotPoint`/`PlotSeries`: Data structures for plot points and series
- `lightCurvePlotRenderer`: Renders plots using gonum/plot, converts to PNG, displays in Fyne canvas
- Supports interactive point selection with coordinate conversion between screen and data space

### Windows API Integration (lines 35-66)
- Direct syscall to user32.dll for window positioning
- Persists window geometry across sessions via Fyne preferences

### External Process Integration
- Spawns IOTAdiffraction.exe and streams its output
- Uses goroutines for stdout/stderr reading with mutex-protected shared state
- Thread-safe UI updates via `fyne.Do()`

### UI Structure
- Fyne v2 application with tabbed interface (4 tabs: Country of origin, Settings, Data, Reports)
- HSplit layout: tabs on left, interactive plot on right
- Menu bar with File, Edit, Help menus

## Key Dependencies

- **Fyne v2**: Cross-platform GUI framework
- **gonum/plot**: Scientific plotting library for light curves

## Generated Files

- `geometricShadow.png`, `diffractionImage8bit.png`, `occultImage16bit.png`: Generated by IOTAdiffraction.exe
- `parameters`: JSON5 configuration file with asteroid/occultation parameters
